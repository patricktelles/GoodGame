GG‑OS v0.4

Sistema Operacional Gamer por Anderson Patrick

Baseado em Arch Linux + XFCE, o GG‑OS é uma distro gamer feita para FPS alto e desempenho máximo em jogos online, até mesmo em PCs fracos.

📥 Ver versões disponíveis

🔥 Componentes Inclusos

  • XFCE4 leve com tema escuro
  • Steam pré-instalada
  • Wallpaper gamer dinâmico
  • Atalhos: Steam e Meus Jogos
  • Pasta /games na Área de Trabalho
  • Scripts de personalização: menu.sh

🚀 Como usar a ISO

  1. Grave com o Rufus
  2. Boot pelo pendrive
  3. Interface XFCE carregada automaticamente
  4. Clique na Steam e comece a jogar!

⚙️ Instalação no HD

Use o instalador do Arch Linux (terminal) ou instale manualmente com:

pacstrap /mnt base base-devel
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt

📋 Requisitos Mínimos

  • CPU: Dual-core 1.6GHz+
  • RAM: 2 GB (4 GB ideal)
  • GPU: compatível com OpenGL 2.0
  • Armazenamento: 8 GB livre
